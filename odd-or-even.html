Problem statement

Get 10 numbers as input from the user and display the given number is odd or even along with the orginal numbers entered.

Assumptions

  • User should input only umbers. No alphabets or special characters are allowed.
  • User should enter numbers in the range between 0 to 1000

Pseudocode

  1. Start the program.
  2. Get the input from the user.
  3. Store the numbers.
  4. Repeat steps 2 and 3 ten times.
  5. Take each number and find the modulus of 2.
  6. if the result is "0" then print it is an even number.
  7. else then print it is an odd number.
  8. Repeat steps 6 and 7 ten times.
